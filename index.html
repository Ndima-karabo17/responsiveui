  • Media query is a CSS technique introduced in CSS3.
  • It uses the @media rule to include a block of CSS properties only if a certain condition is true.

MORE ABOUT MEDIA QUERY

CSS media queries are a powerful tool that allows you to apply CSS styles based on the characteristics of the device or environment rendering the content.
This technique is essential for creating responsive web designs that adapt to different screen sizes, orientations, and other features.

REASON TO USE MEDIA QUERY

Responsive Design
Apply different styles for phones, tablets, desktops, or even TVs
Improved User Experience

WHERE CAN WE USE MEDIA QUERY?

Inside your main CSS file (style.css)
At the bottom of your stylesheet (recommended for overrides)
Or in a separate responsive CSS file
